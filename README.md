## 🔹 Project Overview

## 🔹 Features Implemented

### Block 1:
- DHCP Configuration
- HTTP/HTTPS Server
- DNS Server
- SMTP Email Server

### Block 2:
- LACP EtherChannel
- Port Security

### Block 3:
- VLAN Configuration
- VTP (Server, Client, Transparent)
- Inter-VLAN Routing
- Telnet Access

### Block 4:
- ACL Configuration (Allow/Deny PCs)

### Routing:
- RIP (R1, R2, R3)
- OSPF (R3, R5, R6)
- Redistribution between RIP and OSPF
